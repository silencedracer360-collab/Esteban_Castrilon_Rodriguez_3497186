## ✅ Referencia Rápida de Métodos de Array

Cuando no recuerdes qué método usar:

| Necesito...                 | Método                      |
| --------------------------- | --------------------------- |
| Transformar cada elemento   | `.map()`                    |
| Filtrar por condición       | `.filter()`                 |
| Encontrar uno por condición | `.find()`                   |
| Recorrer sin retornar       | `.forEach()`                |
| Reducir a un valor          | `.reduce()`                 |
| Verificar si alguno cumple  | `.some()`                   |
| Verificar si todos cumplen  | `.every()`                  |
| Incluye un elemento         | `.includes()`               |
| Índice del último           | `.findLastIndex()` ← ES2023 |

## ✅ Referencia Rápida de Métodos de Object

| Necesito...                     | Método                        |
| ------------------------------- | ----------------------------- |
| Array de claves                 | `Object.keys(obj)`            |
| Array de valores                | `Object.values(obj)`          |
| Array de pares `[clave, valor]` | `Object.entries(obj)`         |
| Verificar propiedad propia      | `Object.hasOwn(obj, "prop")`  |
| Copiar / fusionar objetos       | `{ ...obj, ...cambios }`      |
| Crear objeto desde entries      | `Object.fromEntries(entries)` |
