# Ejercicio 01: Planificación de la Solución

## 🎯 Objetivo

Diseñar tu aplicación **antes** de escribir código. Un desarrollador profesional siempre planifica: define el modelo de datos, las funciones necesarias y la estructura de salida antes de implementar.

## ⏱️ Tiempo estimado: 45 minutos

## 📋 Instrucciones

1. Abre `starter/plan.md`
2. Sustituye cada `[placeholder]` con información real de tu dominio asignado
3. Completa los 5 bloques: dominio, modelo de datos, funciones, pseudocódigo y reporte
4. No escribas código JavaScript todavía — este ejercicio es solo diseño en texto

---

## ✅ Lo que debes definir

### 1. Dominio

- Nombre del dominio (tu asignación)
- Nombre del tipo de elemento
- Descripción en una línea

### 2. Modelo de datos

- Mínimo 5 propiedades por objeto
- Al menos 1 numérica, 1 booleana, 1 opcional

### 3. Lista de funciones

- Mínimo 5 funciones
- Nombre claro + qué recibe + qué retorna

### 4. Pseudocódigo del reporte

- Descripción en español de los pasos de `buildReport()`

### 5. Estructura de la salida

- Título
- Estadísticas a mostrar
- Listado de elementos
