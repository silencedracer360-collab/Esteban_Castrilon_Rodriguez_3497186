# 🧪 Prácticas — Semana 10: Proyecto Integrador Etapa 0

Esta semana no introduce sintaxis nueva. Las prácticas están diseñadas para
revisar todos los conceptos vistos en la Etapa 0 (semanas 01–09) antes de
construir el proyecto integrador.

---

## 📋 Ejercicios

| #   | Ejercicio                     | Tipo                 | Tiempo estimado |
| --- | ----------------------------- | -------------------- | --------------- |
| 01  | Planificación de la solución  | Diseño (Markdown)    | 45 min          |
| 02  | Integración — código completo | JavaScript (consola) | 40 min          |

> ⏱️ Tiempo total: ~1.5 horas (bloque de planificación del horario semanal)

---

## 🎯 Objetivo de las prácticas

| Ejercicio                    | Propósito                                                                             |
| ---------------------------- | ------------------------------------------------------------------------------------- |
| `ejercicio-01-planificacion` | Diseñar la solución **antes** de codificar: modelo de datos, funciones y pseudocódigo |
| `ejercicio-02-integracion`   | Verificar que **todas** las semanas 01–09 están integradas en un mismo archivo JS     |

---

## 🚀 Cómo ejecutar el código

```bash
# Ejercicio 02 — starter (código comentado, descomenta sección a sección)
node 2-practicas/ejercicio-02-integracion/starter/index.js

# Ejercicio 02 — solution (todo ejecutando)
node 2-practicas/ejercicio-02-integracion/solution/index.js
```

---

## 📌 Orden recomendado

1. Haz primero **ejercicio-01**: planifica con papel/markdown antes de abrir el editor
2. Luego **ejercicio-02**: descomenta cada sección y verifica que comprendes cada línea
3. Usa la teoría [`01-revision-etapa-0.md`](../1-teoria/01-revision-etapa-0.md) como referencia si tienes dudas
