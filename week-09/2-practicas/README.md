# 🧪 Prácticas — Semana 09: Objetos en JavaScript

Cinco ejercicios guiados que cubren la creación y manipulación de objetos, los métodos `Object.*`, la iteración con `for...in`, el spread operator y los arrays de objetos.

## 📋 Índice de Ejercicios

| #   | Ejercicio                                              | Concepto principal                                     | Tiempo |
| --- | ------------------------------------------------------ | ------------------------------------------------------ | ------ |
| 01  | [Crear y Acceder](./ejercicio-01-crear-acceder/)       | `{}`, acceso punto/corchetes, modificar, anidado       | 25 min |
| 02  | [Métodos de Objeto](./ejercicio-02-metodos-objeto/)    | `Object.keys()`, `Object.values()`, `Object.entries()` | 30 min |
| 03  | [Verificar e Iterar](./ejercicio-03-verificar-iterar/) | `Object.hasOwn()`, operador `in`, `for...in`           | 30 min |
| 04  | [Spread con Objetos](./ejercicio-04-spread-objetos/)   | `{...obj}`, fusionar, actualización inmutable          | 35 min |
| 05  | [Arrays de Objetos](./ejercicio-05-arrays-objetos/)    | `filter`, `find`, `map`, encadenado                    | 40 min |

**Total estimado**: ~160 minutos (2h 40min)

---

## 🚀 Cómo ejecutar

Cada ejercicio tiene su propia carpeta con `starter/index.js` y `solution/index.js`.

```bash
# Ejecutar el starter de cualquier ejercicio
node bootcamp/week-09/2-practicas/ejercicio-01-crear-acceder/starter/index.js

# O navegar al directorio y ejecutar directamente
cd bootcamp/week-09/2-practicas/ejercicio-01-crear-acceder/starter
node index.js
```

---

## 📝 Metodología

Cada ejercicio sigue el mismo flujo:

1. **Leer el README** del ejercicio — comprende el concepto y revisa los ejemplos
2. **Abrir `starter/index.js`** — el código guía ya está escrito, comentado
3. **Descomentar sección por sección** — verificar el resultado en consola
4. **Comparar con `solution/index.js`** solo si tienes dudas

> **Importante**: Los ejercicios son tutoriales guiados — el código ya está escrito.
> Tu tarea es descomentarlo, leerlo con atención y entenderlo.
> El proyecto semanal en `3-proyecto/` sí requiere que implementes desde cero.
