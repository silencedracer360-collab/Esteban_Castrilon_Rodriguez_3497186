# 📚 Ebooks Gratuitos — Semana 08: Arrays

## Recursos Recomendados

### 1. Eloquent JavaScript — Capítulo 4: Data Structures

**Autor**: Marijn Haverbeke
**Enlace**: https://eloquentjavascript.net/04_data.html
**Idioma**: Inglés (versión en español disponible en el sitio)
**Por qué leerlo**: El capítulo 4 es la referencia definitiva sobre arrays y objetos en JavaScript. Cubre creación, acceso, métodos y patrones de uso. Lectura obligatoria para cualquier desarrollador JavaScript.
**Temas relevantes**: Arrays, métodos de iteración, la diferencia entre objetos y arrays.

---

### 2. You Don't Know JS Yet — Get Started (Capítulo 2)

**Autor**: Kyle Simpson
**Enlace**: https://github.com/getify/You-Dont-Know-JS/blob/2nd-ed/get-started/ch2.md
**Idioma**: Inglés
**Por qué leerlo**: Explica cómo JavaScript maneja los valores en memoria, incluyendo la diferencia entre valores primitivos y referencias (clave para entender el problema del spread operator).
**Temas relevantes**: Referencias vs valores, arrays como objetos especiales.

---

### 3. JavaScript.info — Arrays (Capítulo completo)

**Enlace**: https://javascript.info/array
**Idioma**: Inglés (hay traducción al español parcial)
**Por qué leerlo**: Tutorial interactivo muy completo con ejercicios al final de cada sección. Excelente para reforzar lo visto en clase.
**Temas relevantes**: Todos los métodos de array, iteración, performance.

---

### 4. JavaScript.info — Array Methods

**Enlace**: https://javascript.info/array-methods
**Idioma**: Inglés
**Por qué leerlo**: Referencia exhaustiva de todos los métodos de array con ejemplos visuales. Incluye `map`, `filter`, `find`, `findIndex`, `forEach`, `reduce` y más.
**Temas relevantes**: Métodos funcionales, chaining (encadenamiento).

---

> 💡 **Recomendación**: Empieza con Eloquent JavaScript Cap. 4 y JavaScript.info Arrays. Son los más accesibles y cubren todo lo necesario para la semana.
