# Rúbrica de Evaluación — Semana 07: Funciones

## 📊 Criterios de Evaluación (100 puntos)

| #   | Criterio                               | Puntos | Descripción                                                          |
| --- | -------------------------------------- | ------ | -------------------------------------------------------------------- |
| 1   | **El script se ejecuta sin errores**   | 20 pts | `node script.js` completa sin excepciones ni errores de sintaxis     |
| 2   | **Declaración de funciones**           | 15 pts | Al menos 3 funciones declaradas con `function` o arrow function      |
| 3   | **Uso correcto de `return`**           | 15 pts | Las funciones devuelven valores que se usan en el resto del programa |
| 4   | **Arrow functions**                    | 15 pts | Al menos 2 arrow functions `=>` usadas correctamente                 |
| 5   | **Parámetros por defecto**             | 10 pts | Al menos 1 función con parámetro por defecto                         |
| 6   | **Funciones como valores / callbacks** | 15 pts | Al menos 1 función pasada como argumento o asignada a variable       |
| 7   | **Dominio coherente + clean code**     | 10 pts | Nombres descriptivos, dominio aplicado, sin código redundante        |

**Total: 100 puntos**

---

## 📋 Niveles de Desempeño

| Nivel               | Puntos | Descripción                                                                           |
| ------------------- | ------ | ------------------------------------------------------------------------------------- |
| 🥇 **Excelente**    | 90–100 | Cumple todos los criterios; código limpio, nombres descriptivos y comentarios útiles  |
| 🥈 **Bueno**        | 75–89  | Cumple la mayoría de criterios; pequeñas omisiones o inconsistencias menores          |
| 🥉 **Suficiente**   | 60–74  | Cumple los criterios básicos (ejecuta + funciones + return); faltan algunos conceptos |
| ❌ **Insuficiente** | < 60   | No ejecuta, no usa funciones, o no usa `return` para devolver valores                 |

---

## ⚠️ Penalizaciones

| Infracción                                                | Penalización                 |
| --------------------------------------------------------- | ---------------------------- |
| Uso de `var` en lugar de `const`/`let`                    | -10 pts                      |
| Uso de `==` en lugar de `===`                             | -5 pts                       |
| Nombres de funciones no descriptivos (`f1`, `fn`, `func`) | -5 pts                       |
| Código copiado de otro aprendiz                           | -100 pts (calificación de 0) |
| Funciones declaradas pero nunca llamadas                  | -5 pts                       |
| `return` usado pero valor nunca utilizado                 | -5 pts                       |

---

## ✅ Checklist de Entrega

Antes de entregar, verifica:

- [ ] `node script.js` se ejecuta sin errores
- [ ] El script tiene al menos 3 funciones
- [ ] Todas las funciones tienen `return` (excepto las que solo usan `console.log`)
- [ ] Al menos 2 arrow functions
- [ ] Al menos 1 parámetro por defecto
- [ ] Al menos 1 uso de función como valor
- [ ] Nombres de variables y funciones en inglés
- [ ] Comentarios en español
- [ ] El código está adaptado al dominio asignado
- [ ] No se usó `var`
